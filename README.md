# Unifont Single Storey

A Unifont variant with better-looking 'a', 'g', and 'y' characters.


### Modify and Build

1. Clone this repo `git clone https://github.com/Lumine1909/Unifont_Single_Storey.git`.

2. Open wsl, execute `./setup`.

3. Modify `unifont.hex` for what you like. (You may use https://unicucumber.vercel.app/ for editing, and `paste.py` for time-saving).

4. Execute `./build` to build the font, the final file is `out/unifont.ttf`
